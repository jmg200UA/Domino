Programa hecho en Java con Programación Orientada a Objetos en el que se hace uso de Herencia y Polimorfismo.

El objetivo de esta pr´actica es implementar el juego del dominó. Para ello se implementarán las
siguientes clases: Ficha, Tablero, Saco, Jugador, ObjetoNoValidoException, FichaRepetidaException,
SacoIncompletoException, JugadaIncorrectaException, CierreException y PartidaGanadaException.
Además se tiene que implementar la clase Juego, que contendrá un main en el que se jugará
una partida completa leyendo la información de un fichero de texto y escribirá en otro fichero de
texto el resultado de la partida.
